# Var, let and const

1. What is the output of the following code?

```javascript
console.log(number);
var number = 'five'
```

2. What is the output of the following code?

```javascript
let current = 'row';

if (true) {
    let current = 'column'; 
}

console.log(current)
```

3. What is the output of the following code?

```javascript
const array = [];
array[0] = 1;

console.log(array);
```
