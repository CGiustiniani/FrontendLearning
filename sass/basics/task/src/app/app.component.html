Instructions
  • Open the app.component.scss file
  • Inside the stylesheet, declare two variables - $primary and $secondary - and assign them colors of your choice
  • Declare a variable named $base-size and assign a value of 18px to it.
  • Create a new selector for the content section and give it left/right padding of 1 em and set the font size to $base-size.
  • Nest the selector for the heading inside it, give it a color of $primary color, a font size 2.5 times greater than the base size, a bottom border colored $secondary (other settings are arbitrary) and a bottom margin half the size of the $base-size
  • Appropriately nest the list selector for the list and reset the style
  • Implement the hover state of each list element - the element should change the color to $secondary
