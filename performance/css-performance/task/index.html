CSS Tipps & Tricks

1. Reduce render bocking CSS

Add media="print" to the link tag to load it only in print view (<link href="print.css" rel="stylesheet" media="print" />)

2. <link> Preload

Add a preload tag for the file font.woff2 file next to the <title> tag in the <head> (<link href="font.woff2" as="font" type="font/woff2" crossorigin />)

3. CSS property: content-visibility

Add following properties to the .card class in the style.css file:

  • content-visibility: auto;
  • contain-intrinsic-size: 1000px;

4. Media query: prefers-reduced-data

Add following style to the style.css file to remove the custom font for reduced-data devices

@media (prefers-reduced-data: reduce) {
    body {
        font-family: Arial, Helvetica, sans-serif;
    }
}
