Instructions
  • 1) Configure the Http Interceptor (interceptor.service) in the app.module.ts file
  • 2) Add the InterceptorMonitor service to the Http Interceptor
  • 3) Execute the start() method before the request starts and execute stop() after the request has been finalized.
  • 4) Add a header with the key and value "Authentication" to each request.
