Instructions
  • Bind to the Form Fields so the Results represent the form field values
  • Use Angular Currency Pipe so the Currency Result is shown without decimals after comma
  • Use Angulars Date Pipe so the Date Result is shown in this format dd-mm-yyyy
  • Use Angulars Uppercase Pipe so the Text Result is shown in uppercase
  • Create a custom truncate pipe to truncate text longer than 5 letter and show "..." instead
Pipes
Result: {{currencyValue | currency : 'EUR' : 'symbol' : '1.0-0' }}
Result: {{dateValue | date: 'dd.MM.yyyy'}}
Result: {{uppercaseValue | uppercase}}
Result: {{truncateValue | truncate: 5}}
