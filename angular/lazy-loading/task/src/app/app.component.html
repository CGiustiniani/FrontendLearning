Instructions
  • Create a new module named Orders with a component Orders

  • Define routing for that module and its component by defining OrdersRoutingModule in a new file orders-routing.module.ts. See other components for an example.

  • All there is left to do inside a module is to include OrdersRoutingModule in orders module.

  • Open app routing file and find variable routes that instantiates Routes. Using home and customers components as example, inside the instantiation add newly defined orders routing using loadChildren that activates lazy loading for specified route.
