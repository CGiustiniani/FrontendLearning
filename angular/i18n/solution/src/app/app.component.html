Please be aware that Stackblitz does not fully support loading resources like JSON formatted i18n files. For this reason, I provided code to a few demos on GitHub.

There are also other StackBlitz translation examples out there. However, proper i18n translations can't be found on StackBlitz, as loading JSON files (instead of importing them) does not work as mentioned above.
