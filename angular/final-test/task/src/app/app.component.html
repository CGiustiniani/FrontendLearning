Instructions
Service
  • Add a method to retrieve the user from an api via http get (https://randomuser.me/api/).
  • Add a method to store an user into the localstorage, which is called in method 1 after the api call
  • Add a method that returns the user from the localstorage if it exists else calls method 1 and returns the user from the api
Form
  • Retrieve the user by using your UserService (loadProfile)
  • Create a reactive form with title, firstname, lastname, email (createProfileForm)
  • Fill the form fields with the data from the user (updateFormValues)
  • Show the users registration date in the following format (dd.MM.yyy)
  • Add validators to the reactive form, matching the fields errors
  • Use ngIf & ngClass to highlight and show error messages correctly
  • Check if the form is valid on submit (onUpdate)
Component
  • Create a new image component and pass the profile picture url into it
  • Use an image tag and style the picture as a circle
  • Add the image component above the form to show the profile picture
